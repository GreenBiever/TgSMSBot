# SMS bot
### Бот для приёма смс
## Getting started
Для запуска необходим MySQL

1. Скачайте репозиторий и установите зависимости: `pip 
install -r requirements.txt`

2. В файле `default_config.ini` задайте необходимые переменные.
3. Выполните команду `python cli.py` - это обновит данные о доступных для аренды сервисах
3. Выполните `python main.py`


## Поддержка нескольких языков
 - Весь текст, который должен быть переведён, должен быть определён в ```lang_pkg.source_data.source```
 - Для генерации переведённого текста, выполните команду ```python .\generate_lang_packages.py generate```, находясь в директории ```lang_pkg```
 (Необходимо заменить API ключ в файле ```translate.py``` на свой)
 - Для ручой корректировки авоматическгого перевода, измените содержимое файла ```lang_pkg.lang_data.xml```

## Файлы конфигурации
По умолчанию использутся файл `default_config.ini`. Если в корневой папке проекта существует
файл `config.ini`, то будут использоваться значения, указанные в этом файле, 
а файл `default_config.ini` будет игнорироваться.
